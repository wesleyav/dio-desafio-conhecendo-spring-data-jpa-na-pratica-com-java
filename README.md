### Como rodar a aplicação:

No terminal, clone o projeto:

```git
git clone https://github.com/wesleyav/dio-desafio-conhecendo-spring-data-jpa-na-pratica-com-java.git
```

Na raiz do projeto, execute:

```bash
mvn spring-boot: run
```
